# Welcome to free currency converter !

## Demo ---> [link to exchanger](https://marekk871.github.io/Currency_converter/)

In this simple app you can exchange **Euro** to the following currencies :

- USD Dollar
- British Pound
- Chinese Yuan
- Canadian Dollar
- Polish Zloty
